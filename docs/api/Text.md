## Text 
 
Basic Text Component that provides shorthands for sizing,
lineHeight, and fontWeights.
### Props
Name | Type | Default Value | Description
--- | --- | --- | --- 
tiny | bool  |   | 
small | bool  |   | 
medium | bool  |   | 
large | bool  |   | 
giant | bool  |   | 
thin | bool  |   | 
thick | bool  |   | 
bold | bool  |   | 
inverted | bool  |   | 
lineHeight | number  |   |
